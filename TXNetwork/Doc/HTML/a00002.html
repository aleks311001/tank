Простейшая программа

Первое, что вы должны сделать при использовании библиотеки TXNetwork это указать компилятору, что нужно присоединить файл Ws2_32 (который используется Winsock'ом). Делается это просто: в настройках линкера укажите файл Ws2_32 (без разрешения). Настройки линкера находятся в (Code::Blocks)"Settings\Compiler...". Далее выбираете вкладку "Linker Settings", и смотрите в левое окно: Если есть что-то похожее на "Ws2_32", то все хорошо. Иначе нажмите кнопку "Add", и введите это название. Нажмите "ОК". Все, вы готовы использовать TXNetwork.

Библиотека оперирует сокетами. Сокет - абстрактный объект, представляющий точку соединения. Теперь просто. Представьте, что интернет - улица, а компьютеры - дачные участки. Тогда Сокеты будут воротами, через которые дом будет связан через улицу с другими домами.

Сокеты в TXNetwork четко разделены на два вида: Клиент и Сервер. Различие между ними заключается в том, что сервер создается первым.

Каждый компьютер в сети имеет свой адрес(номер) как и длм на улице. Адрес для компьютера в сети называется IP адресом. узнать адрес своего компьютера вы можете командой "ipconfig".

Клиент
TX_SOCKET client = txCreateSocket (TX_CLIENT, "127.0.0.1");
Сервер
TX_SOCKET server = txCreateSocket (TX_SERVER, "127.0.0.1");

Далее будет рассмотрен только код клиента. Для сервера все аналогично, только когда одна часть отправляет, другая принимает.

Отправка и принятие сообщения. Вывод принятого сообщения и завершение программы по нажатию любой клавиши.

txSendTo (client, "Hi! You know about client socket\n", sizeof (STD_STR));
char buf[sizeof (STD_STR)] = "";
txRecvTo (client, buf, sizeof (STD_STR));
printf (buf);
printf ("Ok, but I don't know nothing about My socket...");
getch();
return 0;
}
Код примеров
Далее представлен весь код примеров.

Example1_Client_Code

//{===========================================================================
// $Copyright: (C) _RedMan_ (Vova Yrovsky) <vova@txlib.ru> $
//}===========================================================================
#include "TXLib.h"
#include "TXNetwork.h"
const char STD_STR[] = "Hi! You know about client socket\n";
const int str_len = strlen(STD_STR);
int main()
{
txCreateWindow (1024, 768);
printf ("Hi! \n");
TX_SOCKET client = txCreateSocket (TX_CLIENT, "127.0.0.1");
txSendTo (client, STD_STR, str_len);
char buf[str_len] = "";
txRecvTo (client, buf, , str_len);
printf (buf);
printf ("Ok, but I don't know nothing about My socket...");
getch();
return 0;
}

Example1_Server_Code

//{===========================================================================
// $Copyright: (C) _RedMan_ (Vova Yrovsky) <vova@txlib.ru> $
//}===========================================================================
#include "TXNetwork.h"
#include "TXLib.h"
const char STD_STR[] = "Hi! You know about client socket\n";
const int str_len = strlen(STD_STR);
int main()
{
txCreateWindow (1024, 768);
printf ("Hi! \n");
TX_SOCKET server = txCreateSocket (TX_SERVER, "127.0.0.1");
char buf[str_len] = "";
txRecvTo (server, buf, str_len);
txSendTo (server, STD_STR, str_len);
printf (buf);
printf ("Ok, but I don't know nothing about My socket...");
getch();
return 0;
}
